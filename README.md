# 📊 Customer Churn Analysis Dashboard

## 🚀 Project Overview

This project analyzes customer churn behavior using Power BI.
The goal is to identify patterns that lead customers to leave a subscription-based service.

## 🛠 Tools Used

* Power BI Desktop
* Data Visualization
* Business Analytics

## 📈 Key Insights

* Month-to-Month contracts show higher churn risk.
* Customers with higher monthly charges tend to churn more.
* Long-tenure customers are more loyal.

## 📂 Dataset

Telco Customer Churn Dataset
